**简历在线生成与管理系统**

基于 Spring Boot + Vue3 构建的在线简历编辑平台，支持用户注册登录、简历模块化编辑、多模板实时预览、分享链接发布、管理员后台等完整功能闭环。

**技术栈**：Spring Boot 3.2、MyBatis 3.0、MySQL 8.0、Vue3 + Vite + Pinia + Axios、纯 CSS 模板引擎

**核心功能**：
- 实现基于 Session 的认证拦截器，支持普通用户与管理员双重登录体系，BCrypt 密码加密，管理员可查看全平台用户列表、简历统计（用户数、已发布/草稿数）
- 设计 Ajax 异步加载的响应式 SPA，Vite 代理解决跨域，axios 统一拦截器自动处理认证过期跳转与错误提示
- 设计 5 套不同风格纯 CSS 简历模板（专业/创意/现代/极简/学术），通过分类字段动态匹配，切换模板后右侧 A4 纸实时预览立即生效
- 管理员可在线管理模板（名称、描述、分类、启用/禁用、付费状态），预览图支持本地上传，独立存储服务路径可配置
- 实现 UUID 令牌分享链接生成，支持有效期设置、启用/停用、浏览次数统计，公开页面邮箱和电话自动脱敏
- 编辑器左侧模块化表单（基本信息、工作经历、教育、技能、项目、证书、语言、自定义模块），支持拖拽排序与批量操作
- 独立设计数据库表结构（9 张核心表），MyBatis 动态 SQL 实现多条件筛选查询与关联加载
